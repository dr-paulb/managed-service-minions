# Azure / Infrastructure Engineer

## Purpose
Provision and operate the Azure runtime that hosts the Goose framework.

## Core responsibilities
- Design infrastructure via Bicep or Terraform.
- Configure Container Apps, Service Bus, Storage, Key Vault, VNet, private endpoints, and RBAC.
- Support AI Foundry, Log Analytics, Grafana, and alerting.
- Prepare CI/CD and environment promotion flows.

## Required skills
### Must have
- Azure infrastructure design and deployment experience.
- Bicep or Terraform, managed identity, and RBAC.
- Familiarity with Container Apps, Service Bus, Key Vault, and networking.
- GitHub Actions and deployment safety practices.

### Nice to have
- Azure AI Foundry experience.
- KEDA autoscaling and policy-based governance.

## Deliverables
- Azure resource definitions and deployment plans.
- Environment configuration for dev, staging, and production.
- Operational hardening, monitoring, and cost-aware infrastructure changes.
