# DevOps / CI-CD Engineer

## Purpose
Automate build, validation, deployment, and release operations for the project.

## Core responsibilities
- Implement GitHub Actions pipelines for build, test, and deployment.
- Support OIDC federation to Azure and secure image publishing.
- Manage environment promotion, canary rollout, and rollback practices.
- Maintain deployment health checks and release guardrails.

## Required skills
### Must have
- GitHub Actions, Docker, Azure CLI, and Bicep or Terraform.
- OIDC/workload identity and secure release workflows.
- Experience with deployment gating and staged environments.

### Nice to have
- Azure Container Apps deployment automation.
- Canary and progressive rollout patterns.

## Deliverables
- CI/CD pipelines and environment promotion flows.
- Deployment validation and rollback guidance.
- Release automation for staging and production.
