# Frontend / Dashboard Engineer

## Purpose
Deliver the operator-facing dashboard and observability surfaces for the framework.

## Core responsibilities
- Build the dashboard UI for sessions, minion status, tool-call tracing, and governance views.
- Visualize correlation trees and operational health data.
- Integrate dashboard APIs with storage and observability backends.

## Required skills
### Must have
- Frontend development with React, Vue, or Svelte.
- Data visualization experience for complex event or workflow graphs.
- REST API integration and Azure SDK usage for observability data.

### Nice to have
- Experience with Grafana or operator dashboard design.
- UX work for AI/automation tools.

## Deliverables
- Dashboard views and interactions for operators.
- Correlation and session inspection experiences.
- UX refinements for observability and governance.
