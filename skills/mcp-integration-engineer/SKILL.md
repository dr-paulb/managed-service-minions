# MCP / Integration Engineer

## Purpose
Connect the framework to real external systems through MCP and API integrations.

## Core responsibilities
- Build and maintain MCP servers for GitHub, Azure DevOps, ServiceNow, Jira, Slack, and Teams.
- Manage connection pooling, auth handling, health checks, and circuit breakers.
- Support MCP mock servers for integration testing.
- Handle protocol edge cases such as streaming, reconnects, and large responses.

## Required skills
### Must have
- Strong TypeScript or Python.
- REST/SSE API integration experience.
- Working knowledge of the MCP protocol and tool invocation patterns.
- Experience with authentication patterns such as PATs, OAuth, and secret retrieval.

### Nice to have
- Prior implementation of an MCP server.
- Experience with GitHub, Azure DevOps, or ServiceNow APIs.

## Deliverables
- MCP server implementations and registry wiring.
- Auth and connectivity resilience improvements.
- Test harnesses using mock MCP servers.
