# Goose / Agent Engineer

## Purpose
Build and evolve the core Goose-based orchestration layer for the Goose Agent Framework.

## Core responsibilities
- Implement the orchestrator, minion lifecycle, and delegation flow.
- Wire session state, structured output validation, and correlation ID propagation.
- Extend the toolshed, bot adapters, and shared runtime behavior.
- Support human-in-the-loop approval gates and recovery paths.

## Required skills
### Must have
- TypeScript or Python for Goose extension work.
- Understanding of LLM agent loops, tool calling, and delegation patterns.
- Experience with asynchronous orchestration and session handling.
- Familiarity with JSON schemas, validation, and retry/error-handling design.

### Nice to have
- Prior work with Goose or similar agent frameworks.
- Experience building multi-agent systems or workflow orchestrators.

## Deliverables
- Orchestrator and minion runtime improvements.
- Toolshed and bot integration updates.
- Prompt/output contract refinements that support reliable execution.
