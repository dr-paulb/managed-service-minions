# Product Owner

## Purpose
Define scope, priorities, and delivery readiness for the Goose framework.

## Core responsibilities
- Prioritize which intents, integrations, and workflows matter first.
- Translate user feedback into delivery decisions and roadmap choices.
- Define governance defaults and production acceptance criteria.
- Help decide when the system is ready for wider use.

## Required skills
### Must have
- Product planning and prioritization.
- Understanding of developer workflows such as tickets, PRs, and code review.
- Familiarity with Slack, Teams, and AI-assisted team workflows.

### Nice to have
- Experience with AI or agent products.
- Working knowledge of enterprise developer tooling.

## Deliverables
- Prioritized backlog and scope guidance.
- Acceptance criteria for MVP and release readiness.
- Feedback loops for quality, UX, and operational improvements.
