# QA / Test Engineer

## Purpose
Make the framework reliable through automated and manual verification.

## Core responsibilities
- Build unit, integration, prompt-quality, E2E, and chaos tests.
- Maintain mock MCP servers and realistic test scenarios.
- Add regression coverage for production incidents and workflow regressions.

## Required skills
### Must have
- Strong testing skills in TypeScript or Python.
- Integration test design and mock-server experience.
- Familiarity with a test framework such as Jest or pytest.

### Nice to have
- Chaos or performance testing experience.
- Prompt evaluation and regression test design background.

## Deliverables
- Test suites for core workflows and security controls.
- MCP mock infrastructure and scenario packs.
- Regression coverage for known failure paths.
